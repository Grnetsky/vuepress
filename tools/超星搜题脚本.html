          # 超星搜题脚本

          # 简介

          超星搜题脚本主要用于超星学习通，中国大学mooc等主流学习软件的搜题功能，针对不同的搜题环境包含了五种搜题模式，以下进行一一介绍

          # 安装

          1. 下载超星搜题脚本源码
          2. 打开文件ctrl+A全选，ctrl+C复制
          3. 打开浏览器点击油猴脚本如图（若未安装请先安装油猴脚本 (opens new window)）
          4. 点击添加脚本

          5. 删除视图中所有的代码

          6. 将刚复制的源码ctrl+V粘贴在页面中

          7. ctrl+s保存代码即可（默认为自动打开，为保证正常运行，可点击管理面板确认脚本已成功打开）

          # 使用方法

          1. 进入测验题，屏幕左边会出现开启脚本按钮，点击按钮即可开始

          2. 点击按钮后，按提示输入 数字 （一定要为数字，否则脚本出错）需要的模式
          3. 此时脚本会自动进行题目匹配搜索，如图
          4. 稍等片刻答案会自动渲染到页面上

          # 模式介绍

          # 1. 超星作业模式

          该模式根据超星学习通量身打造的模式，支持新老版本的超星作业题搜索，可搜索课后作业和考试题

          # 2. mooc测验题模式

          该模式为mooc打造，专门针对mooc的测验题搜索，需要说明的是由于mooc的升级换代，题库题更新较慢，导致mooc部分题目无法搜到，这将会在后续版本不断优化

          # 3. 自定义节点题搜索

          该模式可用于任何地方，该模式提高了题目搜索的灵活性，属于 开发者 使用的模式（普通用户若不会使用，可使用其他模式）
          使用方法如下

          1. 进入题目页面,按 F12 键
          2. 点击Elements选项
          3. 点击Elements左边第一个小图标
          4. 点击第一个题目
          5. 复制题目对应的类名，并找到题目所在位置的索引数
          6. 点击开启脚本，选择模式3
          7. 输入刚选择的类名，用选择器的方式输入
          8. 输入题目所在位置的索引数
          9. 回车确认后稍等片刻，答案即可搜索到

          # 4. 手动输入题目查询

          该模式用于拓展题目搜索的灵活性，可以定义搜索题目 只需在输入框中输入包含题目的数组即可
          例如['题目1','题目2','题目3'...] 确认后点击提交，等待显示答案

          # 5. 网点自助查询

          该模式会自动跳转到第三方网点，输入对应题目即可自助查询答案（该模式需要关闭弹窗拦截）
