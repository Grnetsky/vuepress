# mark down核心用法

## （一）段落格式



### 标题



​	用 # 代表标题级数 ##表示二级标题



### 字体

*斜线*：用 \*文本*或\_斜体_

__粗体__:  用 \__粗体__

___粗斜体__: 用\_______粗斜体___



### 分割线

用三个以上的***



### 删除线

~~删除线~~: \~~文字~~



###  下划线

<u>下划线</u>：用\<u>文字</u>



### 脚注

脚注[^脚注]

[^脚注]: 这是脚注在需要注明的内容外加上\[^内容]





## （二）列表

### 无序列表

+ 文字前使用 *号即可

### 有序列表

1. 使用数字加[^.]号即可

### 列表嵌套

1. 父层
   1. 子层
   2. 只需要在父层下按下tab剪即可
